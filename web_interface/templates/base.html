# Usage Instructions The Test Stand Dashboard is a web application that allows you to control the test stand remotely. ## Running Unit Tests Running unit tests is currently managed through PIO Remote. If the PIO remote status is running (see below), you can run unit tests by clicking the "Run Unit Tests" button, simply click the "Remote test" button in VSCode to push and execute code. ## Webcam ** NOTE: The webcam is currently not supported. ** If the webcam is connected, you can view the live feed by navigating to the website section in the sidebar. NOTE: the webcam feed may not be available if it is being used by the unit test script.
